# Merasa Resto - Android App

Aplikasi katalog menu restoran modern yang dibangun menggunakan **Jetpack Compose**. Aplikasi ini dirancang untuk memberikan pengalaman pengguna yang mulus dalam menjelajahi menu makanan dan profil restoran.

## Fitur Utama
- **Home Screen**: Tampilan beranda minimalis dengan logo dan sambutan hangat.
- **Daftar Menu**: Katalog menu lengkap dengan kategori.
- **Detail Menu**: Informasi mendalam tentang setiap hidangan.
- **Manajemen Profil**: Pengaturan profil restoran dan preferensi tema.
- **Dark Mode Support**: Mendukung tema gelap dan terang secara dinamis.

## Teknologi yang Digunakan
- **Kotlin**: Bahasa pemrograman utama.
- **Jetpack Compose**: Untuk membangun UI secara deklaratif.
- **Navigation Compose**: Navigasi antar layar.
- **PreferenceManager**: Untuk penyimpanan data lokal dan pengaturan tema.

## Cara Menjalankan
1. Clone repository ini.
2. Buka di **Android Studio Ladybug** atau versi terbaru.
3. Jalankan di emulator atau perangkat fisik.
